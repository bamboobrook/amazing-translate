Amazing Translate MVP

Configuration and Translation Demo

This page shows the exact workflow after loading dist/ as an unpacked Chrome extension.

API Key setup

The real extension stores the key in Chrome extension local storage. It is not committed to Git and is never injected into webpages.

How to call the features

  1. Open the extension popup and click 设置.
  2. Choose DeepSeek or 智谱 GLM Coding, paste the API Key, set languages, and click 保存设置.
  3. Click 测试当前服务商 to verify the configured endpoint.
  4. Open any readable page and click 翻译当前页 from the popup or the floating page button.
  5. Select text and use the context menu item Amazing Translate：翻译选中文本.
  6. Focus a text box and use Amazing Translate：翻译输入内容; replacement only happens after clicking 替换为译文.

Reading Sample

Modern browser extensions can improve reading workflows without sending every page to a remote service automatically. A manual translation flow gives the reader control over cost, privacy, and timing.

The extension scans visible article-like blocks, sends compact batches to the configured model, and inserts bilingual translations under the original text.
